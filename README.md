📌 Project Overview

This project involves analyzing sales data using Excel to understand customer behavior, product performance, and business trends. The goal was to transform raw data into meaningful insights through an interactive dashboard.

🛠 Tools Used

Microsoft Excel

Pivot Tables

Data Cleaning Techniques

Data Visualization (Charts & Dashboard Design)



📂 Dataset

Sales transaction data (orders, quantity, revenue, customer details)


📈 Key Insights

Sales fluctuated across 13 weeks, with peak performance in the middle period,
Customers purchasing 2–3 items formed the largest segment,
T-shirts, Jeans, and Sneakers were the top-selling products,
Most purchases were made through digital channels (Apps & Websites),
Delivery time was mostly within 2–3 days


📊 Dashboard Features

KPI summary (Orders, Quantity, Revenue, Rating, Delivery Time),
Sales trend over time,
Customer purchase behavior analysis,
Product performance breakdown,
Geographic distribution of customers,
Shipment duration analysis
